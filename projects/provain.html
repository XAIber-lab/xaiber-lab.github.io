About the project

ProVAIN builds network attack-and-defense demonstration scenarios set inside fictional military network topologies, used to walk through how an attacker's path maps onto MITRE ATT&CK techniques and how corresponding defenses map onto D3FEND countermeasures.

Network topologies are generated in a range of structures (ring, mesh, zig-zag, star) and enriched with CVEs across a diverse set of ATT&CK techniques, so that each scenario reflects a plausible, technically-grounded attack surface rather than a purely illustrative diagram.

A companion GUI tool supports both local inference (via Ollama) and cloud inference (via the Google Gemini API), with automated repair of generated network-topology data and CVE enrichment built into the generation pipeline.
